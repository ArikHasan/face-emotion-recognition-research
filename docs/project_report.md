# Project Report

## Title

Real-Time Facial Emotion Recognition Using Computer Vision and Deep Learning

## Abstract

Facial emotion recognition is an important application of computer vision and artificial intelligence. This project presents a real-time facial emotion recognition system using Python, OpenCV, TensorFlow, and the FER library. The system captures live video from a webcam, detects human faces, and predicts emotions such as happy, sad, angry, fear, surprise, disgust, and neutral. The goal of this project is to build a simple, practical, and student-friendly emotion recognition system that can be used for academic learning, research practice, and portfolio development.

## Introduction

Human facial expressions are one of the most natural ways of showing emotions. In modern artificial intelligence, facial emotion recognition is used in human-computer interaction, smart education, mental health monitoring, user experience analysis, and security systems.

This project focuses on detecting emotions from real-time webcam input. Instead of training a deep learning model from scratch, this project uses a pre-trained emotion recognition model through the FER library. This makes the system easier to run on a low-configuration laptop.

## Problem Statement

Understanding human emotions automatically from facial expressions is a challenging task in computer vision. Manual emotion analysis is time-consuming and subjective. A real-time automated system can help detect emotions quickly from live video input.

The main problem addressed in this project is:

How can a simple computer vision-based system detect human facial emotions in real time using a webcam?

## Objectives

The main objectives of this project are:

1. To capture real-time video from a webcam.
2. To detect human faces from video frames.
3. To recognize facial emotions using a pre-trained deep learning model.
4. To display the predicted emotion with confidence score.
5. To build a simple and low-cost academic project using Python.

## Technologies Used

- Python
- OpenCV
- TensorFlow
- FER Library
- NumPy
- Webcam

## Methodology

The system follows a step-by-step process:

1. Webcam Input
2. Frame Capture
3. Face Detection
4. Emotion Recognition
5. Confidence Score Calculation
6. Real-Time Output Display
