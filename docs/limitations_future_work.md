# Limitations and Future Work

## Limitations

Although this project can detect facial emotions in real time, it has some limitations.

1. The accuracy may decrease in poor lighting conditions.
2. Low-quality webcam video can affect face detection and emotion prediction.
3. The system may not detect emotion correctly if the face is partially covered.
4. Facial masks, glasses, or side-facing positions may reduce performance.
5. The system uses a pre-trained model instead of a custom-trained model.
6. Emotion prediction may not always match the real emotional state of a person.
7. The project may run slowly on very low-configuration laptops.

## Future Work

This project can be improved in the future in several ways.

1. A custom CNN model can be trained using the FER2013 dataset.
2. A graphical user interface can be added using Tkinter or PyQt.
3. Emotion detection results can be saved in a CSV file.
4. The system can be improved to detect multiple faces at the same time.
5. A web-based version can be developed using Flask or Django.
6. An Android application version can be developed.
7. Model accuracy can be compared with other deep learning models.
8. Better face detection methods can be added for improved performance.

## Conclusion

The current version of the project is simple, lightweight, and suitable for academic learning and research practice. With future improvements, it can become more accurate, user-friendly, and useful for real-world applications.
