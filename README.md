
# Real-Time Face Emotion Recognition System

This project is a real-time facial emotion recognition system built using Python, OpenCV, TensorFlow, and the FER library. The system captures video from a webcam, detects human faces, and predicts facial emotions such as happy, sad, angry, fear, surprise, disgust, and neutral.

## Project Title

**Real-Time Facial Emotion Recognition Using Computer Vision and Deep Learning**

## Purpose of the Project

The main purpose of this project is to develop a simple and practical machine learning-based system that can detect human facial emotions in real time using a webcam. This project can be used for academic research, portfolio building, and basic computer vision learning.

## Features

- Real-time webcam-based face emotion detection
- Face detection using computer vision
- Emotion prediction with confidence score
- Simple Python implementation
- Low-cost and student-friendly project
- Suitable for research, academic report, and GitHub portfolio

## Detected Emotions

The system can detect the following emotions:

- Angry
- Disgust
- Fear
- Happy
- Sad
- Surprise
- Neutral

## Technologies Used

- Python
- OpenCV
- TensorFlow
- FER Library
- NumPy
- Webcam

```text
Webcam Input

     ↓
```
